# 📊 Portfolio Risk & Volatility Monitoring Dashboard

##  Overview
This project analyzes portfolio performance and financial risk using Python and Tableau. It focuses on measuring asset behavior, evaluating risk, and visualizing insights through an interactive dashboard.

---

##  Features
- Log return calculation for performance analysis  
- Rolling volatility (30-day window)  
- Correlation matrix for asset relationships  
- Monte Carlo simulation (10,000 iterations)  
- Risk metrics: Value at Risk (VaR), Volatility, Sharpe Ratio  
- Interactive Tableau dashboard  

---

## Tech Stack
- Python (Pandas, NumPy)  
- Tableau  
- CSV / Excel  

---

##  Workflow
1. Data collection and cleaning  
2. Log return calculation  
3. Risk analysis (volatility & correlation)  
4. Monte Carlo simulation  
5. Portfolio metrics calculation  
6. Dashboard visualization  

---

##  Output Files
- `returns.csv` – Daily returns  
- `rolling_volatility.csv` – Risk over time  
- `correlation_matrix.csv` – Asset correlation  
- `mc_results.csv` – Simulation results  
- `portfolio_summary.csv` – Key metrics
